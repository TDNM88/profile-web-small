Usage Instructions for Tab 1: Invoice Extractor

Step 1:

Upload an invoice document in PDF, JPG, PNG, or JPEG format using the "Upload Invoice" section.

Step 2:

Click the "Extract Contents" button to extract raw text from the uploaded document. The extracted text will appear in the "Contents for next step" box.

Step 3:

Click the "Next Step" button to process the extracted text and add the information to the table below. You can repeat Steps 1-3 with additional files to append more data to the table.

Step 4:

When finished, click "Export to CSV & Reset" to download all extracted data as a CSV file. The table will be reset after exporting, and you can start processing new documents.

Usage Instructions for Tab 2: Transaction Extractor

Step 1:

Upload a bank statement document in PDF format using the "Upload Bank Statement (PDF)" section.

Step 2:

Click the "Extract Transactions" button to extract transaction data from the uploaded PDF. The extracted data will be processed and displayed in the table below.

Step 3:

If you have additional PDF files to process, repeat Steps 1-2. Each new file's data will be appended to the existing table.

Step 4:

When finished, click "Export to CSV & Reset" to download all extracted transaction data as a CSV file. The table will be reset after exporting, allowing you to start processing new documents.
