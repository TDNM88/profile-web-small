Ứng dụng Tạo Nội dung và Video

Dưới đây là hướng dẫn chi tiết từng bước để sử dụng các chức năng của ứng dụng:

  1. Bước 1: Mở tab "Tạo Nội dung".
  2. Bước 2: Nhập yêu cầu nội dung vào ô "Nhập yêu cầu nội dung".
  3. Bước 3: (Tùy chọn) Tải lên file kèm theo nếu bạn muốn sử dụng nội dung từ file đó.
  4. Bước 4: Chọn loại nội dung từ danh sách thả xuống "Chọn loại nội dung".
  5. Bước 5: Nhấn nút "Tạo Nội dung" để hệ thống tạo ra nội dung dựa trên yêu cầu của bạn.
  6. Bước 6: Chỉnh sửa nội dung nếu cần trong ô "Nội dung tạo ra".
  7. Bước 7: Nhấn nút "Xác nhận nội dung" để xác nhận nội dung đã chỉnh sửa. Hệ thống sẽ tự động tạo file DOCX và chuyển sang bước tạo âm thanh.
  8. Bước 8: Tải xuống file DOCX và file âm thanh từ các nút tải xuống tương ứng.
  1. Bước 1: Mở tab "Tạo Âm thanh".
  2. Bước 2: Nhập văn bản bạn muốn chuyển đổi thành âm thanh vào ô "Nhập văn bản để chuyển đổi".
  3. Bước 3: Chọn giọng đọc từ danh sách thả xuống "Chọn giọng đọc".
  4. Bước 4: Nhấn nút "Tạo Âm thanh" để hệ thống tạo ra file âm thanh.
  5. Bước 5: Nghe âm thanh tạo ra và tải xuống file âm thanh từ nút tải xuống.
  1. Bước 1: Mở tab "Tạo Video".
  2. Bước 2: Chọn file âm thanh từ máy tính của bạn bằng cách nhấn vào "Chọn file âm thanh".
  3. Bước 3: Chọn hình ảnh từ danh sách thả xuống "Chọn hình ảnh" để sử dụng trong video.
  4. Bước 4: Nhấn nút "Tạo Video" để hệ thống tạo ra video từ âm thanh và hình ảnh đã chọn.
  5. Bước 5: Xem video tạo ra và tải xuống file video từ nút tải xuống.
  • Đảm bảo rằng bạn đã cấu hình đúng các API keys cần thiết trước khi sử dụng ứng dụng.
  • Kiểm tra định dạng file khi tải lên để đảm bảo hệ thống có thể xử lý đúng cách.
  • Nếu gặp lỗi, hãy kiểm tra lại các bước và đảm bảo rằng tất cả các thông tin đầu vào đều chính xác.
