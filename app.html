Ứng dụng Tạo Nội dung và Video

Dưới đây là hướng dẫn chi tiết từng bước để sử dụng các chức năng của ứng dụng:

  1. Mở tab "Tạo Nội dung".
  2. Nhập yêu cầu nội dung vào ô "Nhập yêu cầu nội dung".
  3. (Tùy chọn) Tải lên file kèm theo (PDF hoặc DOCX) nếu bạn muốn sử dụng nội dung từ file đó.
  4. Chọn loại nội dung từ các tùy chọn có sẵn (podcast, giới thiệu, triết lý sống, Phổ biến kiến thức thống kê).
  5. Nhấn nút "Tạo Nội dung" để hệ thống tạo ra nội dung dựa trên yêu cầu của bạn.
  6. Chỉnh sửa nội dung nếu cần trong ô "Nội dung tạo ra".
  7. Nhấn nút "Xác nhận nội dung" để xác nhận nội dung đã chỉnh sửa. Hệ thống sẽ tự động tạo file DOCX và file âm thanh.
  8. Tải xuống file DOCX và file âm thanh từ các nút tải xuống tương ứng.
  1. Mở tab "Tạo Âm thanh".
  2. Nhập văn bản bạn muốn chuyển đổi thành âm thanh vào ô "Nhập văn bản để chuyển đổi".
  3. Chọn giọng đọc từ danh sách thả xuống "Chọn giọng đọc" (alloy, echo, fable, onyx, nova, shimmer).
  4. Nhấn nút "Tạo Âm thanh" để hệ thống tạo ra file âm thanh.
  5. Nghe âm thanh tạo ra và tải xuống file âm thanh từ nút tải xuống.
  1. Mở tab "Tạo Video".
  2. Nhập kịch bản cho video vào ô "Nhập kịch bản cho video".
  3. (Tùy chọn) Nhập đường dẫn đến dataset ảnh nếu bạn muốn sử dụng ảnh từ dataset.
  4. Tích vào ô "Sử dụng dataset ảnh" nếu bạn muốn sử dụng dataset ảnh.
  5. Nhấn nút "Tạo hình ảnh từ kịch bản" để hệ thống tạo ra các hình ảnh phù hợp với kịch bản.
  6. Kiểm tra các hình ảnh và nội dung được tạo ra.
  7. Chọn file âm thanh từ máy tính của bạn bằng cách nhấn vào "Chọn file âm thanh".
  8. Nhấn nút "Tạo Video" để hệ thống tạo ra video từ âm thanh và hình ảnh đã chọn.
  9. Xem video tạo ra và tải xuống file video từ nút tải xuống.
  • Đảm bảo rằng bạn đã cấu hình đúng các API keys cần thiết (GROQ_API_KEY, OPENAI_API_KEY, Pexels_API_KEY) trước khi sử dụng ứng dụng.
  • Khi tải lên file kèm theo trong phần Tạo Nội dung, chỉ hỗ trợ định dạng PDF và DOCX.
  • Trong phần Tạo Video, nếu không sử dụng dataset ảnh, hệ thống sẽ tự động tìm kiếm ảnh từ Pexels.
  • Quá trình tạo video có thể mất một chút thời gian, đặc biệt là khi xử lý nhiều hình ảnh.
  • Nếu gặp lỗi, hãy kiểm tra lại các bước và đảm bảo rằng tất cả các thông tin đầu vào đều chính xác.
